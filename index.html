Quick-Start Audit in 5 Minutes

The full code is in the GitHub repository. Basic workflow:

  1. Query your database for images missing alt text using PHP or WP-CLI
  2. Export results to CSV and review
  3. Add alt text manually or use wp eval-file to bulk update
  4. Verify using the same query — should be zero (or near-zero)
